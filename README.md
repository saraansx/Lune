### ◈ The Vision

Lune is a refined Spotify client that prioritizes aesthetics and performance. It strips away the noise of traditional players, leaving only a clean, focused interface for your music library. Built for those who appreciate both sound and space.

### ✧ Features

✦ **Pure Minimalism** ─ An interface that stays out of your way.
✦ **Desktop Optimized** ─ Full Electron-based desktop integration.
✦ **Visual Harmony** ─ Cohesive dark aesthetics across every component.
✦ **Rich Presence** ─ Built-in Discord integration to showcase your library.
✦ **Library Virtualization** ─ Smooth scrolling even with thousands of tracks.

<br />

### ◈ Getting Started

To build and run Lune locally on your machine:

```bash
# 1. Clone the project
git clone https://github.com/yourusername/lune.git

# 2. Install dependencies
npm install

# 3. Launch the development environment
npm run dev
```
